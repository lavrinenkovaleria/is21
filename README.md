# Анализатор CSV

## Состав команды
- Белоус Виктория (модуль Парсинг CSV: загрузка файла, вывод заголовков и первых строк и модуль Фильтрация данных: отбор строк по значению в столбце.)
- Лавриненко Валерия (модуль Фильтрация данных: отбор строк по значению в столбце и модуль Статистика: подсчет среднего, мин/макс для числового столбца.)


## Инструкция по запуску
1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать окружение и установить зависимости: `pip install -r requirements.txt`
4. Запустить: `python main.py`
